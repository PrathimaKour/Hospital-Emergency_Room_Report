## Hospital Emergency Room Dashboard (Excel Project)

## Project Overview
This project is an interactive **Excel Dashboard** designed to analyze the performance of a Hospital Emergency Room.
It is inspired by the work of **Satish Dhawle** and focuses on key operational metrics.

---

## Objective
To monitor and improve hospital efficiency by analyzing:
  * Patient volume
  * Waiting time
  * Admission rates
  * Department referrals
  * Patient satisfaction

---

## Key Features
  * Monthly & Year-wise Analysis (2023 vs 2024)
  * Total Patients Count
  * Average Waiting Time
  * Patient Satisfaction Score
  * Admission vs Non-Admission Analysis
  * Age Group Distribution
  * Gender Analysis
  * *Department Referral Breakdown
  * On-time vs Delayed Service Analysis

---

## Tools & Skills Used
  * Microsoft Excel
  * Pivot Tables
  * Pivot Charts
  * Slicers
  * Data Cleaning
  * Dashboard Design
  * Data Visualization

---

## Dataset Details
  * Data includes patient records for Emergency Room visits
  * Fields used: Date, Age Group, Gender, Wait Time, Admission Status, Department Referral, Satisfaction Score
  * Data has been cleaned and structured using Excel before visualization

---

## How to Use the Dashboard
  1. Open the Excel file
  2. Use **Year slicer (2023 / 2024)** to filter data
  3. Select **Month buttons** to analyze trends
  4. Interact with charts to explore patient insights

---

## Insights
  * 61% patients were attended on time
  * Equal admission vs non-admission rate (50%)
  * Highest patients belong to age group **30–44**
  * Most referrals were not required (majority “None”)
  * Slightly higher male patient count compared to female
